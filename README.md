# Currency Converter

A React app for converting between different currencies, using http://exchangeratesapi.io.

## Prerequisites

- Node >= 8.10
- npm >= 5.6

## Installation

`npm install`

## Running in Development

`npm start`

This should open in a tab in your browser, but you may alternatively access it at http://localhost:3000. The port may be different if you have other applications already occupying that port - it will be displayed in your shell.
